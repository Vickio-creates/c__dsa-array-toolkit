# DSA Array Toolkit (C++)

## Overview
This project is a collection of basic array operations implemented in C++.
It is designed to practice fundamental Data Structures and Algorithms (DSA) concepts such as iteration, conditionals, and memory handling.

## Features
Find the maximum value in an array
Find the minimum value in an array
Reverse an array
Count even numbers in an array
User input for dynamic array creation

## Concepts Practiced
Arrays
Functions
Loops (`for`)
Conditional statements (`if`)
Basic problem-solving techniques

## Technologies Used
C++
Standard Library (`iostream`)

## Future Improvements
Add sorting algorithms (Bubble Sort, Selection Sort)
Implement searching algorithms (Linear Search, Binary Search)
Improve memory efficiency (in-place operations)
Convert to menu-driven program

# DSA Practice

A collection of Data Structures and Algorithms exercises written in C++.

## Files

### dsa-array-toolkit.cpp
Array operations using index notation:
- Print array
- Find max and min
- Sum and average
- Reverse array
- Count even numbers
- Dynamic user input

### pointerToolkit.cpp
Array operations using pointer arithmetic:
- Print array using `*(arr + i)`
- Find max, min, sum
- Reverse array
- Swap values
- Count even numbers

### pointers-arrays.cpp
Combining pointers and arrays:
- `findMaxPointer` — returns address of max element
- `swapPointers` — swaps two elements using addresses

### bubble-sort.cpp
Sorting algorithm implementation:
- Bubble sort — compares and swaps neighbors
- Sorts array from smallest to largest
- Time complexity: O(n²)
  
## Key Concept
`arr[i] == *(arr + i)` — arrays and pointers are connected at memory level.
